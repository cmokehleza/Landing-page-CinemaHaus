# CinemaHaus Landing Page

This project is a landing page for CinemaHaus, a modern movie theater experience. It is built using HTML and designed to be visually appealing and user-friendly.

## Features
- Responsive design
- Movie showcase
- Easy navigation

## Getting Started
Simply open `index.html` in your browser to view the landing page.
